# Paimonade 🍹

A General use Genshin Impact Discord Bot built with WOK Discord Command Handler thing

## Usage

1. Clone this repo

2. `npm install`

3. create a config.json file in the parent directory containing your discord bot token
```
{
  "token": "your token"
}
```

4. `npm start`
